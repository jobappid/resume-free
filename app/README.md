# Resume Builder (Free, No AI)

- No accounts
- No saving / share links
- Step-by-step wizard (simple questions)
- 3 templates (Classic / Modern / Minimal)
- Download PDF via browser print-to-PDF

## Run locally
```bash
cd app
npm install
npm run dev
```

## Cloudflare Pages
- Root directory: `app`
- Build command: `npm run build`
- Output directory: `dist`
